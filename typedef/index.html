    public AuthRouter-ResetToken(token: string): function source

    This endpoint is hit by an email to reset a user password This endpoint is hit first in the sequence

    • Endpoint: /auth/api/v2/reset/:token

    • Verb: GET

    • OnFailure: Redirects to the login screen with an error in query string

    • OnSuccess: Redirects to the forgot-redirect page to change password

    Params:

    NameTypeAttributeDescription
    token string

    A string that contains the HEX code/Reset token of a lost account
